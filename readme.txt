Bước 1 Download cài đặt Opencart

Bước 2 Download module trên trang  http://vnpayment.vn    

Bước 3 Giải nén file đã download

Bước 4 Copy thư mục admin,catalog paste vào thư mục gốc của website

Bước 5 vào trang quản trị của website vào Menu=>Extensions=>Payments tìm đến module vnpayment click install

Bước 6 click Edit Câp nhật cấu hình đăng ký tại VNPAY

Bước 7 Thành công ra thực hiện thanh toán qua vnpay

IPN https://xxx/index.php?route=extension/payment/vnpay_payment/vnpay_ipnurl
